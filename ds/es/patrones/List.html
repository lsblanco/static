El componente patrón list permite crear un listado de etiquetas.

Tipos de lista Pendiente ​

La propiedad type permite definir diferentes tipos de lista, siendo los valores permitidos none (por defecto), disc, circle y square.
None
  • Piña
  • Naranja
Disc
  • Piña
  • Naranja
Circle
  • Piña
  • Naranja
Square
  • Piña
  • Naranja
Ver código
vue
<template>
  <!--None-->
  <ea-list :data="dataTipo" />

  <!--Disc-->
  <ea-list :data="dataTipo" type="disc" />

  <!--Circle-->
  <ea-list :data="dataTipo" type="circle" />

  <!--Square-->
  <ea-list :data="dataTipo" type="square" />
</template>
ts
<script setup lang="ts">
  import { EaList } from '@c/ea';
  const dataTipo = [
    { key: '1', label: 'Piña' },
    { key: '2', label: 'Naranja' }
  ];
</script>

Colores de la lista Pendiente ​

Las propiedades bgColor y border permiten personalizar el fondo y el borde del contenedor de la lista. Siendo los valores posibles para el fondo transparent (por defecto), white, y gray, Siendo los valores posibles para el borde none (por defecto), primary, y gray.
None
  • Coche
  • Moto
  • Casa
Primary
  • Coche
  • Moto
  • Casa
Secondary
  • Coche
  • Moto
  • Casa
Tertiary
  • Coche
  • Moto
  • Casa
Ver código
vue
<template>
  <!--None-->
  <ea-list :data="dataColores"/>

  <!--Primary-->
  <ea-list :data="dataColores" bg-color="white" borderColor="primary"/>

  <!--Secondary-->
  <ea-list :data="dataColores" bg-color="gray" borderColor="gray"/>

  <!--Tertiary-->
  <ea-list :data="dataColores" bg-color="white" borderColor="gray"/>   
</template>
ts
<script setup lang="ts">
  import { EaList } from '@c/ea';
  const dataColores = [
    { key: '1', label: 'Coche' },
    { key: '2', label: 'Moto' },
    { key: '3', label: 'Casa' }
  ];
</script>

La propiedad title permite incluir un título en la lista.
Con título

  • Moto
  • Casa
Sin título (por defecto)

  • Moto

</template>

<script setup lang="ts">

    { key: '1', label: 'Coche' },

Dimensiones de la lista Pendiente ​

Las propiedades width y height permiten personalizar las dimensiones de la lista con unidades px, em, %. Si no utilizamos esta propiedad la lista se adaptará en alto y ancho al contenedor que contiene la lista.
Por defecto, se adapta al contenedor
  • Coche
  • Moto
  • Casa
50%
  • Coche
  • Moto
  • Casa
250px
  • Coche
  • Moto
  • Casa
Ver código
vue
<template>
  <!--Por defecto, se adapta al contenedor-->
  <ea-list :data="dataColores" bgColor="gray" borderColor="gray"></ea-list>

  <!--50%-->
  <ea-list width="50%" :data="dataColores" bgColor="gray" borderColor="gray"></ea-list>

  <!--250px-->
  <ea-list width="250px" :data="dataColores" bgColor="gray" borderColor="gray"></ea-list>
</template>
ts
<script setup lang="ts">
  import { EaList } from '@c/ea';
  const dataColores = [
    { key: '1', label: 'Coche' },
    { key: '2', label: 'Moto' },
    { key: '3', label: 'Casa' }
  ];
</script>

Contenido personalizado Pendiente ​

Con los distribuidores de contenidos o también llamados slots podemos personalizar nuestro componente con mucha flexibilidad. Disponemos de dos distribuidores de contenidos personalizables dentro del componente ea-list, contenido en la parte superior o cabecera, e inferior o pié.
Contenido en la cabecera
    En el contenido superior o header podemos poner cualquier elemento html.
Contenido en el centro o default
    En el contenido central o default podemos poner cualquier elemento html.
Contenido en el pié
    En el contenido inferior o pié podemos poner cualquier elemento html.
Todos los contenidos
    Contenido en la parte superior o header
    Contenido en la parte central o default
    Contenido en la parte inferior o footer
Ver código
vue
<template>
  <!--Contenido en la cabecera-->
  <ea-list :data="dataColores">
    <section slot="header" class="t-border-dashed-primary-1">
      En el contenido superior o header podemos poner cualquier elemento <b>html</b>.
    </section>
  </ea-list>
  <!--Contenido en el centro o default-->
  <ea-list :data="dataColores">
    <section class="t-border-dashed-primary-1">
      En el contenido central o default podemos poner cualquier elemento <b>html</b>.
    </section>
  </ea-list>
  <!--Contenido en el pié-->
  <ea-list :data="dataColores">
    <section slot="footer" class="t-border-dashed-primary-1">
      En el contenido inferior o pié podemos poner cualquier elemento <b>html</b>.
    </section>
  </ea-list>
  <!--Todos los contenidos-->
  <ea-list>
    <section class="t-border-dashed-primary-1" slot="header">Contenido en la parte superior o header</section>
    <section class="t-border-dashed-primary-1" slot="default">Contenido en la parte central o default</section>
    <section class="t-border-dashed-primary-1" slot="footer">Contenido en la parte inferior o footer</section>
  </ea-list>
</template>
ts
<script setup lang="ts">
  import { EaList } from '@c/ea';
  const dataColores = [
    { key: '1', label: 'Coche' },
    { key: '2', label: 'Moto' },
    { key: '3', label: 'Casa' }
  ];
</script>
