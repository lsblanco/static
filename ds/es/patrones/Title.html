Componente patrón Title.

Títulos Pendiente ​

Con las propiedades string title y subtitle, como su nombre indica podemos personalizar el titulo e introducir un nuevo subtitulo.
Título (por defecto)

Titular de página

Título personalizado

Título personalizado

Con subtitulo

Titular de página

Subtítulo personalidado
Ver código
vue
<template>
  <!--Título (por defecto)-->
  <ea-title></ea-title>

  <!--Título personalizado -->
  <ea-title title="Título personalizado" ></ea-title>

  <!--Con subtitulo-->
  <ea-title subtitle="Subtítulo personalidado" ></ea-title>
</template>
ts
<script setup lang="ts">
  import { EaTitle } from '@zurich-es-npm/ea-front-web-ui';
</script>

Distribuidores de contenido Pendiente ​

Con los distribuidores de contenidos o también llamados slots podemos personalizar nuestro componente con mucha flexibilidad. Disponemos de tres distribuidores de contenidos personalizables dentro del componente ea-title, contenido en la parte central, superior e inferior.
Contenido en la parte central

Titular de página

Contenido en la parte superior

Titular de página

Contenido en la parte inferior

Titular de página

Todos los contenidos

Titular de página

Ver código
vue
<template>
  <!--Contenido en la parte central-->
  <ea-title>
    <section class="t-border-dashed-primary-1">
      En el contenido central podemos poner cualquier elemento <doc-b>html</doc-b>.
    </section>
  </ea-title>
  <!--Contenido en la parte superior -->
  <ea-title>
    <section class="t-border-dashed-primary-1" slot="up">
      En el contenido superior podemos poner cualquier elemento <doc-b>html</doc-b>.
    </section>
  </ea-title>
  <!--Contenido en la parte inferior-->
  <ea-title>
    <section class="t-border-dashed-primary-1" slot="down">
      En el contenido inferior podemos poner cualquier elemento <doc-b>html</doc-b>.
    </section>
  </ea-title>
  <!--Todos los contenidos-->
  <ea-title>
    <section class="t-border-dashed-primary-1" slot="up">Contenido en la parte superior</section>
    <section class="t-border-dashed-primary-1" slot="default">Contenido en la parte central</section>
    <section class="t-border-dashed-primary-1" slot="down">Contenido en la parte inferior</section>
  </ea-title>
</template>
ts
<script setup lang="ts">
  import { EaTitle } from '@zurich-es-npm/ea-front-web-ui';
</script>

Casos de uso Pendiente ​

El componente ea-title es un componente muy versátil que se puede utilizar en diferentes casos de uso. A continuación, se muestran algunos ejemplos de cómo se puede utilizar este componente en diferentes situaciones.

Mantenimiento de pólizas

Nº póliza: 0000G1234567
Ver código
vue
<template>
  <!--Caso de uso con acciones-->
  <ea-title title="Mantenimiento de pólizas" subtitle="Nº póliza: 0000G1234567">
    <section class="d-display-flex d-justify-flex-end">
      <ea-button class="m-r-16 d-hidden-md-and-down" type="secondary" size="medium">Más acciones</ea-button>
      <ea-button class="m-r-16 d-hidden-md-and-down" type="primary" size="medium">Acción principal</ea-button>
      <ea-dropdown 
          trigger="click"
          placement="bottom-end"
          @command="onHandleCommand">
          <span slot="label">
            <ea-button class="d-hidden-md-and-down" type="secondary" icon="z-more-solid" size="medium" square></ea-button>
            <ea-button class="d-hidden-lg-and-up" type="primary" icon="z-more-solid" size="medium" square></ea-button>
          </span>
          <ea-dropdown-item command="nuevo">Nuevo</ea-dropdown-item>
          <ea-dropdown-item command="editar">Editar</ea-dropdown-item>
          <ea-dropdown-item command="consultar">Consultar</ea-dropdown-item>
          <ea-dropdown-item command="eliminar">Eliminar</ea-dropdown-item>
          <ea-dropdown-item command="mas" class="d-hidden-lg-and-up">Más Acciones</ea-dropdown-item>
          <ea-dropdown-item command="principal" class="d-hidden-lg-and-up">Acción principal</ea-dropdown-item>
      </ea-dropdown>
    </section>
  </ea-title>
</template>
ts
<script setup lang="ts">
  import { EaTitle, EaButton, EaDropdown, EaDropdownItem } from '@zurich-es-npm/ea-front-web-ui';

  const onHandleCommand = (command) => {
    if(command === 'editar') {
      console.log('Edita...');
    } else {
      console.log(command)
    }
  }
</script>

Caso de uso con navegación Pendiente ​

En este caso de uso, el componente ea-title muestra un navegador de páginas o migas de pan en Desktop y un botón de navegar hacia atrás en mobile.

Mantenimiento de pólizas

Nº póliza: 0000G1234567
Ver código
vue

  <ea-title title="Mantenimiento de pólizas" subtitle="Nº póliza: 0000G1234567">

        <ea-breadcrumb-item :to="{ path: '/' }">inicio</ea-breadcrumb-item>
        <ea-breadcrumb-item :to="{ path: '/gestion' }">gestión</ea-breadcrumb-item>
        <ea-breadcrumb-item :to="{ path: '/lista' }">lista</ea-breadcrumb-item>
        <ea-breadcrumb-item>detalle</ea-breadcrumb-item>
      </ea-breadcrumb>
    </section>
  </ea-title>
</template>
