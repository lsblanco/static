Tipos de formato Terminado ​

La propiedad format permite definir el formato del valor, siendo los valores posibles string (por defecto), number, currency, date, boolean y link.
Con la propiedad config podemos cambiar los valores iniciales siguientes:
js
  config = { 
    locale: 'es-ES', // por defecto en number, currency y date.
    currency: 'EUR', // por defecto en currency.
    grouping: false, // por defecto en  number. 
    grouping: true,  // por defecto en currency. 
    decimals: 0,     // por defecto en number y currency. 
  }
