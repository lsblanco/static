FloatButton ​

Componente
Beta

El componente EaFloatButton permite lanzar una acción al pulsar un botón flotante que aparece en una página determinada.

Uso básico ​

El componente de botón flotante aparece situado en la parte inferior derecha de la pantalla o contenededor, y nos permite realizar una acción al pulsarlo.

Botón flotante en página o contenedor ​

La propiedad target permite asignar el botón flotante a un contenedor, estando por defecto asignada a la página si se omite dicha propiedad.

vue
<template>
  <!-- Botón flotante en la página (por defecto) -->
  <ea-button type="secondary" @click="handleClick">{{ text }}</ea-button>
  <ea-float-button :visible="visible" />

  <!-- Botón flotante en un contenedor -->
  <div class="container__basic wp-300 hp-200">
    <ea-float-button 
      target=".container__basic"
    />
    <div>
      <div v-for="item in getArray()">Mensaje {{item}}</div>
    </div>
  </div>
</template>
ts
<script setup lang="ts">
  import { ref } from 'vue';
  import { EaFloatButton, EaButton } from '@zurich-es-npm/ea-front-web-ui';

  const visible = ref(false);
  const text = ref('Mostrar botón flotante en la página');

  function getArray() {
    const arr = []
    for(let i=0; i <= 400; i++) {
      arr.push(i);
    }
    return arr;
  };

  function handleClick() {
    visible.value=!visible.value;
    text.value = visible.value ? 'Ocultar botón flotante en la página': 'Mostrar botón flotante en la página';
  }
</script>

Posicionamiento del botón ​

Con las propiedades right y bottom, podemos personalizar la posición del botón flotante en la pantalla o contenedor. Por defecto estos valores son de 16 para right y de 16 para bottom. Con la propiedad booleana center, podemos centrar el botón en el contenedor. Por defecto el valor de esta propiedad es false.
Posicionamiento por defecto
Centrado
Posicionamiento personalizado
Ver código
vue
<template>
  <!-- Posicionamiento por defecto -->
  <div class="container__pos1">
    <ea-float-button
      target=".container__pos1"
    />
  </div>

  <!-- Centrado -->
  <div class="container__pos2">
    <ea-float-button
      target=".container__pos2"
      center
    />
  </div>
  
  <!-- Posicionamiento personalizado -->
  <div class="container__pos3">
      <ea-float-button
      target=".container__pos3"
      :right="40"
      :bottom="40"
    />
</template>

Tipo de acción ​

La propiedad type permite cambiar el tipo de acción del botón si se utiliza un icono, siendo primary (por defecto) o secondary los valores permitidos.
Primary (por defecto)
Secondary

      type="secondary"
