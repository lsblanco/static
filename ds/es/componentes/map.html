Map ​

Componente
Alpha

El componente ea-map permite a los usuarios encontrar localizaciones en un mapa y varios modelos de configuración.

Uso básico del mapa Rev Doc ​

El funcionamiento básico del mapa es simplemente con la etiqueta ea-map, aparaceciendo el mapa centrado en la posición en la que se encontramos el usuario actualmente.
Ver código
vue
<template>
  <!-- Uso básico del mapa -->
  <ea-map></ea-map>
</template>
ts
<script setup lang="ts">
  import { EaMap } from '@zurich-es-npm/ea-front-web-ui'
</script>

Geolocalización Pediente ​

Con la propiedad geolocation activamos la geolocalización situando el mapa en la posición actual en la que nos encontramos, siendo los valores posibles true (por defecto) y false.
Mapa con geolocalización activada
Mapa con geolocalización desactivada

Posición actual del usuario ​

Al activar la geolocalización podemos mostrar un icono con nuestra posición exacta con la propiedad current-position, siendo sus valores posibles true (por defecto), o false.
Icono con la posición actual del usuario (por defecto)
Sin icono

Personalización del icono ​

Con la propiedad current-position-icon podemos personalizar el icono que muestra la posición actual de usuario, introduciendo la ruta del icono situada en nuestra aplicación.

Geolocalizarme usando el método geoLocation ​

Como particularidad, puedes usar la herramienta del navegador para geolocalizarte tu mismo usando el método geoLocation. Esto permite, escuchar la posición de la geolocalización del navegador agregando un marcador automáticamente a la vez que hacer zoom en el mapa hacia nuestra posición actual. Para esto debemos desactivar la geolocalización que se envía como parámetro al mapa geolocation = false.
Ver código
vue
<template>
  <!-- Mapa con geolocalización activada -->
  <ea-map></ea-map>

  <!-- Mapa con geolocalización desactivada -->
  <ea-map :geolocation="false"></ea-map>
</template>
ts
<script setup lang="ts">
  import { EaMap, EaButton } from '@zurich-es-npm/ea-front-web-ui'
</script>

Tamaño del mapa Pediente ​

Con la propiedad width y height podemos fijar el ancho y alto del mapa con medidias css, y en caso de no utilizar ambas propiedades el mapa se adaptará a las medidas del contenedor que contiene el mapa.
Mapa con medidas fijas de 300px de alto y ancho
Mapa que se adapta a las medidas de su contenedor
Mapa que ocupa el 50% de la pantalla
Ver código
vue
<template>
  <!-- Mapa con medidas fijas de 300px de alto y ancho -->
  <ea-map
    width="300px"
    height="300px"
  >
  </ea-map>

  <!-- Mapa que se adapta a las medidas de su contenedor -->
  <ea-map></ea-map>

  <!-- Mapa que ocupa el 50% de la pantalla -->
  <ea-map
    width="50vw"
    height="50vh"
  >
  </ea-map>
</template>
ts
<script setup lang="ts">
  import { EaMap } from '@zurich-es-npm/ea-front-web-ui'
</script>

Zoom del mapa Pediente ​

Con la propiedad númerica zoom podemos acercar o alejar el mapa, siendo la valores permitidos del 1 al 22, y el 8 el valor por defecto.
Zoom por defecto (8)
Zoom medio (12)
Zoom más ampliado (22)
Zoom más alejado (1)
Ver código
vue
<template>
  <!-- Zoom por defecto (8) -->
  <ea-map height="300px"></ea-map>

  <!-- Zoom medio (12) -->
  <ea-map :zoom="12" height="300px"></ea-map>

  <!-- Zoom más ampliado (22) -->
  <ea-map :zoom="22" height="300px"></ea-map>

  <!-- Zoom más alejado (1) -->
  <ea-map :zoom="1" height="300px"></ea-map>
</template>
ts
<script setup lang="ts">
  import { EaMap } from '@zurich-es-npm/ea-front-web-ui'

Abrir marcador Pediente ​

Con el método openPopup mostramos el cuadro de diálogo de cada punto informativo.

Con la propiedad infoWindowsOpts podemos agregar las opciones disableAutoPan, maxWidth, y minWidth, como describimos a continuación:

  • disableAutoPan: Permite inhabilitar el desplazamiento del mapa al abrir el marcador.
  • maxWidth: Agrega el tamaño máximo que puede tener el cuadro de diálogo.
  • minWidth: Agrega el tamaño mínimo que puede tener el cuadro de diálogo.
Cuadro de diálogo

Agregar o eliminar un marcador Pediente ​

Con el método addMarkerByCoordinates podemos agregar un nuevo marcador al mapa. El marcador es un elemento más de los puntos que agregamos al mapa mediante la propiedad points. Este método devuelve un nuevo marcador de tipo google.maps.marker.AdvancedMarkerElement.

El objeto points que pasamos como párametro al método addMarkerByCoordinates es el siguiente:

    {
      id: this.id, //(Opcional) Identificador para abrir el modal con openMarkerById.
      lat: this.lat, //latitud del marcador
      lng: this.lng, //Longitud del marcador
      urlIcon: 'images/components/map/point.svg',(Opcional) Url del icono del marcador
      group: this.group //(Opcional) Nombre del grupo del marcador
    }
  • Con el método removeMarker podemos eliminar un marcador. Como parámetro se utiliza el objeto del marcador de tipo google.maps.marker.AdvancedMarkerElement
Cuadro de diálogo
Ver código
vue
<template>
  <!-- Cuadro de diálogo -->
  <ea-map
    ref="map"
    :zoom="8"
    :current-position="false"
    :points="points"
    :geolocation="false"
    height="300px">
  </ea-map>
</template>
ts
<script setup lang="ts">
  import { EaMap, EaButton } from '@zurich-es-npm/ea-front-web-ui'
  import { ref, onMounted } from 'vue';

  const markers = ref([]);
  const lat = ref(40.71226);
  const lng = ref(-4.18934);
  const points = ref([
    {
      id: 1,
      lat: 40.63054, 
      lng: -4.000554,
      urlIcon: '/images/components/map/point.svg',
    },
    {
      id: 2,
      lat: 40.59324, 
      lng: -4.14280,
      urlIcon: '/images/components/map/point2.svg',
    },
    {
      id: 3,
      lat: 40.71225, 
      lng: -4.18933,
    }
  ]);

  const handleButtonAddMarker = () => {
    lat.value += 0.02;
    lng.value += 0.02;
    const newMarker = {
      lat: lat.value,
      lng: lng.value,
      urlIcon: '/images/components/map/point.svg',
    };
    const marker = map.value.addMarkerByCoordinates(newMarker);
    markers.value.push(marker);
  }

  const handleButtonRemoveMarker = () => {
    markers.value.forEach(marker => map.value.removeMarker(marker));
    lat.value = 40.71226;
    lng.value = -4.18934;
  }
</script>

Marcadores Cluster en el mapa Pediente ​

Con la propiedad objeto clusterMarker podemos agrupar los marcadores. Este objeto tiene dos propiedades default y group. - La propiedad Default tiene como valores true o false, siendo el valor true el que activa la agrupación. - La propiedad Group tiene como valor un objeto con las propiedades clave/valor, siendo clave el nombre de la agrupación y valor una clase del sistema de diseño para definir el color: 
  group: {
    '1': 't-color-primary-dark-1',
    '2': 't-color-gray-65',
    '3': 't-color-secondary-10',
  }
El nombre del grupo anteriormente definido debe coincidir con el nombre de la propiedad group en el objeto points que establece los marcadores del mapa: 
  points: [{
    lat: 40.63054, 
    lng: -4.000554,
    urlIcon: 'images/components/map/point.svg',
    group: '1' // Group key.
  },
  {
    lat: 40.59324, 
    lng: -4.14280,
    urlIcon: 'images/components/map/point.svg',
    group: '2' // Group key.
  }
  ...
Además, pueden existir marcadores agrupados y no agrupados en el mismo mapa. Para no agrupar marcadores simplemente no agregaremos la propiedad group en los marcadores del objeto points.
Sin marcadores agrupados (por defecto)
Marcadores agrupados

Eventos Pediente ​

  • Mediante el evento bounds podemos obtener las coordenadas del recuadro del mapa, tal que si el mapa detecta un movimiento, este responderá devolviendo las coordenadas del cuadro del mapa que estamos viendo.

  • Mediante el evento openPopup podemos mostrar un cuadro de diálogo haciendo clic en un marcador.

openPopup ofrece tanto el dato del marcador al cual se hace clic como un métoco callback al cual le tenemos que enviar una cadena de texto de lo que queremos mostrar en el cuadro de diálogo.
bounds
Desplaza el mapa para ver nuevos resultados
Resultado bounds
openPopup
clic en un marcador
Resultado openPopup
