Select ​

Componente
Beta

El componente select permite seleccionar valores entre un conjunto de opciones.

Uso básico ​

Con la propiedad value podemos establecer el valor del select.
Sin value
Con value
Ver código
vue
<template>
  <!-- Sin value-->
  <ea-select>
    <ea-option label="Option 1" value="1"/>
    <ea-option label="Option 2" value="2"/>
    <ea-option label="Option 3" value="3"/>
  </ea-select>

  <!-- Con value-->
  <ea-select value="Option 2">
    <ea-option label="Option 1" value="1"/>
    <ea-option label="Option 2" value="2"/>
    <ea-option label="Option 3" value="3"/>
  </ea-select>
</template>
ts
<script setup>
  import { EaSelect, EaOption } from '@zurich-es-npm/ea-front-web-ui';
</script>

Uso básico con v-model ​

Con la propiedad v-model podemos establecer el enlace a datos.
Valor seleccionado
Valor seleccionado: ninguno
Valor asignado
Valor asignado: 2
Ver código
vue
<template>
  <!--Valor seleccionado-->
  <ea-select v-model="modelSelect">
    <ea-option label="Option 1" value="1"/>
    <ea-option label="Option 2" value="2"/>
    <ea-option label="Option 3" value="3"/>
  </ea-select>
  <span>Valor seleccionado: {{ modelSelect || 'ninguno'}}</span>

  <!--Valor asignado-->
  <ea-select placeholder="Select" v-model="modelAssign">
    <ea-option label="Option 1" value="1"/>
    <ea-option label="Option 2" value="2"/>
    <ea-option label="Option 3" value="3"/>
  </ea-select>
  <span>Valor asignado: {{modelAssign}}</span>
</template>

<script setup lang="ts">
  import { ref } from 'vue';

  const modelSelect = ref(null);
  const modelAssign = ref('2');
</script>
ts
<script setup>
  import { EaSelect, EaOption } from '@zurich-es-npm/ea-front-web-ui';
</script>

Tipologías ​

La propiedad type indica el tipo de ea-select disponible, siendo simple (valor por defecto) y multiple los valores permitidos. ea-select en su formato simple permite seleccionar una única opción, y en su formato múltiple permite seleccionar una, varias o todas las opciones, entre un conjunto de valores disponibles.
Selección simple
Selecciona una opción
Selección múltiple
Selecciona una opción
Ver código
vue
<template>
  <!-- Selección simple -->
  <ea-select placeholder="Selecciona una opción">
    <ea-option label="Option 1" value="1"/>
    <ea-option label="Option 2" value="2"/>
    <ea-option label="Option 3" value="3"/>
  </ea-select>

  <!-- Selección múltiple -->
  <ea-select 
    type="multiple" 
    placeholder="Selecciona una opción"
  >
    <ea-option label="Option 1" value="1"/>
    <ea-option label="Option 2" value="2"/>
    <ea-option label="Option 3" value="3"/>
  </ea-select>
</template>

<script setup>

Selecciona una opción
