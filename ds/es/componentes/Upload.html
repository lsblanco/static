Upload ​

Componente
Beta

El componente upload permite al usuario subir uno o varios archivos seleccionándolos o arrastrándolos.

Tipología ​

Mediante la propiedad drag podemos indicar el tipo de componente para cargar archivos, siendo los valores posibles, true (por defecto) arrastrando, y false haciendo clic.
Carga de archivos arrastrándolos (por defecto)
Suelta tu archivo aquí o haz clic para cargar
Solo archivos [formato] con un tamaño menor de [tamaño máximo]
    Carga de archivos haciendo clic
    Solo archivos [formato] con un tamaño menor de [tamaño máximo]
      Ver código
      vue
      <template>
  <!-- Carga de archivos arrastrándolos (por defecto) -->
  <ea-upload />

  <!-- Carga de archivos haciendo clic -->
  <ea-upload :drag="false" />
</template>
      ts
      <script setup>
  import { EaUpload } from '@zurich-es-npm/ea-front-web-ui';
</script>

                                  Límite, tamaño y tipo de archivos ​

                                  Mediante las propiedades númerica limit, maxFileSize, y accept podemos especificar el número máximo de Limite de archivos. Siendo los valores por defecto 10 archivos, 9 mb de tamaño y todos los tipos de archivos.
                                  Por defecto
                                  Solo archivos [formato] con un tamaño menor de [tamaño máximo]
                                    Con límite de 2 archivos, 50kb y jpeg y png
                                    Solo archivos jpg/png con un tamaño menor de 50kb.
                                      Ver código
                                      vue
                                      <template>
  <!-- Por defecto -->
  <ea-upload :drag="false" />

  <!-- Con límite de 2 archivos, 50kb y jpeg y png -->
  <ea-upload
    :drag="false"
    :limit="2"
    :maxFileSize="50000"
    accept="image/jpeg,image/png"
    description="Solo archivos jpg/png con un tamaño menor de 50kb."
    error="El archivo no es correcto. Solo formatos jpg/png con un tamaño menor de 500kb"
  />
</template>

                                              Borrado de archivos Terminado ​

                                              Mediante el método clearFiles, podemos borrar todos los archivos seleccionados.
                                              Solo archivos [formato] con un tamaño menor de [tamaño máximo]
                                                Ver código
                                                vue
                                                <template>
  <ea-upload
    ref="uploadRef"
    :drag="false"
    :limit="2"
    :maxFileSize="500000"
  >
  </ea-upload>
  <ea-button type="secondary"  @click="handleButtonClick">Borrar Archivos</ea-button>
</template>
                                                ts
                                                <script setup>
  import { EaUpload, EaButton, $eaMessage } from '@c/ea';
  import { useTemplateRef } from 'vue';
  import { EaUpload } from '@zurich-es-npm/ea-front-web-ui'
  
  const uploadRef = useTemplateRef('uploadRef');
  const handleButtonClick = () => {
    uploadRef.value.clearFiles();
    $eaMessage({
      message: 'Lo archivos se han borrado',
      type: 'success',
      duration: 2000
    });
  }
</script>
