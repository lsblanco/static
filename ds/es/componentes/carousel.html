Carousel ​

Componente
Beta

El carousel es un elemento interactivo que muestra una serie de imágenes consecutivas, donde el usuario puede navegar de izquierda a derecha y viceversa.

Indicadores ​

La propiedad indicatorPosition determina dónde se encuentran los indicadores de paginación en el carousel, siendo los valores permitidos, inside (por defecto), outside y none.
Dentro (por defecto)
1
2
3
Fuera
1
2
3
Ninguno
1
2
3
Ver código
vue
<template>
  <!-- Dentro (por defecto) -->
  <ea-carousel >
    <ea-carousel-item v-for="item in 3" :key="item">
      <div :class="slideClass(item)">{{ item }}</div>
    </ea-carousel-item>
  </ea-carousel>
  
  <!-- Fuera -->
  <ea-carousel 
    indicator-position="outside"
  >
    <ea-carousel-item v-for="item in 3" :key="item">
      <div :class="slideClass(item)">{{ item }}</div>
    </ea-carousel-item>
  </ea-carousel>

  <!-- Ninguno -->
  <ea-carousel 
    indicator-position="none"
  >
    <ea-carousel-item v-for="item in 3" :key="item">
      <div :class="slideClass(item)">{{ item }}</div>
    </ea-carousel-item>
  </ea-carousel>
</template>
ts
<script setup lang="ts">
  import { EaCarousel, EaCarouselItem } from '@zurich-es-npm/ea-front-web-ui';

</script>

Modo automático ​

Con la propiedad autoplay se enlazan las diapositivas de modo automático, siendo false (por defecto), o true, los valores permitidos.
False (por defecto)
1
2
3
True
1
2
3
Ver código
vue
<template>
  <!-- False (por defecto) -->
  <ea-carousel>
    <ea-carousel-item v-for="item in 3" :key="item">
      <div>{{ item }}</div>
    </ea-carousel-item>
  </ea-carousel>
  
  <!-- True -->
  <ea-carousel :autoplay="true">
    <ea-carousel-item v-for="item in 3" :key="item">
      <div>{{ item }}</div>
    </ea-carousel-item>
  </ea-carousel>
</template>

Con la propiedad interval definimos el intervalo de tiempo en el que aparecen las diapositivas en modo automático, siendo 5000 milisegundos el valor establecido por defecto.
