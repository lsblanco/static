DatePicker ​

Componente
Beta

El componente DatePicker permite a los usuarios elegir una fecha, o un rango de fechas.

Uso básico ​

Con la propiedad value podemos establecer el valor del componente.
Ver código
vue
<template>
  <ea-date-picker
      :value="new Date()"
    ></ea-date-picker>
</template>
ts
<script setup lang="ts">
  import { EaDatePicker } from '@zurich-es-npm/ea-front-web-ui';
</script>

Uso básico con v-model ​

La propiedad v-model permite enlazar el valor del componente con la vista.
Valor enlazado: Wed Feb 18 2026 08:54:05 GMT+0100 (Central European Standard Time)
Ver código
vue
<template>
  <ea-date-picker
    v-model="model"
  ></ea-date-picker>
  Valor enlazado: {{ model }}
</template>
ts
<script setup lang="ts">
  import { EaDatePicker } from '@zurich-es-npm/ea-front-web-ui';

  const model = ref(new Date());
</script>

Formatos de fecha ​

Con la propiedad format podemos configurar el formato de la fecha, siendo sus valores posibles eu (por defecto) y us.

Nota Importante

El valor introducido en la propiedad value o vmodel tiene que coincidir con el formato de fecha establecido para Eu: dd/mm/aaaa o Us: mm/dd/yyyy

Formato "eu" (por defecto) dd/mm/aaaa
Formato "us" mm/dd/yyyy
Ver código
vue
<template>
  <!-- Eu -->
  <ea-date-picker
    v-model="modelFormat1"
    format="eu"
  ></ea-date-picker>

  <!-- Us -->
  <ea-date-picker
    v-model="modelFormat2"
    format="us"
  ></ea-date-picker>  
</template>
ts
<script setup lang="ts">
  import { EaDatePicker } from '@zurich-es-npm/ea-front-web-ui';

  const modelFormat1 = ref(new Date());
  const modelFormat2 = ref(new Date());
</script>

Placeholder ​

Con la propiedad placeholder podemos añadir un texto cuando el componente no tiene todavía un valor. Por defecto se muestra el formato de fecha a introducir dependiendo del tipo de ea-date-picker, y el formato del idioma seleccionado mediante la propiedad format.
Placeholder por defecto
Placeholder con texto
Ver código
vue
<template>
  <!-- Placeholder por defecto -->
  <ea-date-picker />

  <!-- Placeholder con texto -->
  <ea-date-picker 
    placeholder="Selecciona una fecha ..."
  />
</template>
ts
<script setup lang="ts">

</script>

Deshabilitado y solo lectura ​

La propiedad disabled permite permite deshabilitar el componente date-picker, y con la propiedad readonly mostramos el componente en modo lectura, siendo en ambas propiedades sus valores posibles true y false (por defecto).
Disabled
Read Only
Ver código
vue

    :value="new Date()"
    disabled
  ></ea-date-picker>

  <!-- Read Only -->
  <ea-date-picker
    :value="new Date()"
    readonly
  ></ea-date-picker>

Tipología ​

Con la propiedad type podemos utilizar diferentes tipos del componente ea-date-pickers, siendo los valores posibles date (por defecto), datetime, year y month.
Date (por defecto)
Datetime
Year

  <!-- Month -->

    type="month"
    :value="new Date()"
  ></ea-date-picker>

Rangos ​

También se puede utilizar otros tipos con la propiedad type para definir rangos, siendo los valores posibles daterange, datetimerange y monthrange.
Daterange
-
Datetimerange
-
Monthrange
-
Ver código
vue
<template>
  <!-- Daterange -->
  <ea-date-picker
    type="daterange"
    start-placeholder="Fecha inicio ..."
    end-placeholder="Fecha fin ..."
    :value="dateRange"
  ></ea-date-picker>

  <!-- Datetimerange -->
  <ea-date-picker
    type="datetimerange"
    start-placeholder="Fecha inicio ..."
    end-placeholder="Fecha fin ..."
    :value="dateRange"
  ></ea-date-picker>
  
  <!-- Monthrange -->

    type="monthrange"
    start-placeholder="Fecha inicio ..."
    end-placeholder="Fecha fin ..."
    :value="dateRange"
  ></ea-date-picker>
</template>

<script setup lang="ts">
  import { EaDatePicker } from '@zurich-es-npm/ea-front-web-ui';
  import { ref } from 'vue';

  const dateRange = ref([new Date(), new Date()]);
</script>

Picker Options: Accesos directos ​

Modificando la propiedad shortcuts de la propiedadpickerOptions podemos generar opciones de acceso directo. Cabe notar que shortcuts es un array de objetos {text, onClick}
Ver código
vue
<template>
  <ea-date-picker
    v-model="value"
    :pickerOptions="pickerOptions"
    placeholder="Selecciona una fecha ..."
  ></ea-date-picker>
</template>
ts
<script setup lang="ts">
  import { EaDatePicker } from '@zurich-es-npm/ea-front-web-ui';
  import { ref, computed } from 'vue';
  
  const value = ref('');
  const pickerOptions = computed(() => {
    return {
      shortcuts: [
        {
          text: 'Ayer',
          value: (picker) => {
            const date = new Date();
            date.setDate(date.getDate() - 1);
            return date;
          }
        },
        {
          text: 'Hoy',
          value: (picker) => {
            return new Date();
          }
        },
        {
          text: 'Mañana',
          value: (picker) => {
            const date = new Date();
            date.setDate(date.getDate() + 1);
            return date;
          }
        }
      ]
    };
  });
</script>

Picker Options: disabledDate ​

Con la propiedad disabledDate de la propiedad pickerOptions se puede indicar si una o varias fechas deben de estar deshabilitadas. Se ha de pasar una función con una fecha como parámetro y devolver un booleano. En el ejemplo, por ejemplo, se están deshabilitando todas las fechas que sean posteriores a la fecha actual
Ver código
vue
<template>
  <ea-date-picker
    :value="new Date()"
    placeholder="Selecciona una fecha ..."
    :pickerOptions="pickerOptionsDisabled"
  ></ea-date-picker>
</template>
ts
<script setup lang="ts">
  import { EaDatePicker } from '@zurich-es-npm/ea-front-web-ui';
  
  const pickerOptionsDisabled = computed(() => {
    return {
      disabledDate: (date) => date.getTime() > Date.now(),
    };
  });
</script>

Picker Options: firstDayOfWeek ​

Con la propiedad firstDayOfWeek de la propiedadpickerOptions podemos indicar el primer dia de la semana. Lo valores aceptados son los números entre 1 y 7, siendo 1 el valor por defecto
Ver código
vue
<template>
  <ea-date-picker
    :value="new Date()"
    placeholder="Selecciona una fecha ..."
    :pickerOptions="{
      firstDayOfWeek: 3,
    }"
  ></ea-date-picker>
</template>
ts
<script setup lang="ts">
  import { EaDatePicker } from '@zurich-es-npm/ea-front-web-ui';
</script>

Picker Options: onPick ​

A través de la prop onPick de la propiedadpickerOptions podemos indicar un callback a ejecutar cada vez que la fecha seleccionada cambia. Se trata de un función cuyo parámetro es un objeto {maxDate, minDate}. Sólo para types daterange y datetimerange.
-
Ver código
vue
<template>
  <ea-date-picker
    v-model="dateRange"
    type="daterange"
    placeholder="Selecciona una fecha ..."
    :pickerOptions="{
      onPick: (date) => {
        console.log(`onPick: ${JSON.stringify(date)}`);
      }
    }"
  ></ea-date-picker>
</template>
ts
<script setup lang="ts">
  import { EaDatePicker } from '@zurich-es-npm/ea-front-web-ui';
</script>
