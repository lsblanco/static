Saludo Terminado ​

Con la propiedad greeting podemos incluir el texto del saludo al iniciar el chat. La propiedad greeting permite definir el texto del saludo, siendo su valor ¡Hola! ¿Cómo puedo ayudarte? .
Saludo inicial
Saludo personalizado
Ver código
vue
<template>
  <!-- Sin saludo -->
  <ea-chat />

  <!-- Con saludo -->
  <ea-chat
    greeting="Hola, ¿en qué puedo ayudarte?"
  />
</template>
ts
<script setup lang="ts">
  import { EaChat } from '@zurich-es-npm/ea-front-web-ui';
</script>

Usuario Terminado ​

Con las propiedades user y userIcon indicamos las iniciales del nombre del usuario, el icono o una imagen.
Sin usuario
Con iniciales de usuario
Con icono de usuario
Con imagen de usuario
Ver código
vue
<template>
  <!-- Sin usuario -->
  <ea-chat :data="chatData" />
  
  <!-- Con iniciales de usuario -->
  <ea-chat 
    user="Ea"
    :data="chatData" 
  />

  <!-- Con icono de usuario -->
  <ea-chat 
    userIcon="user-accident-solid"
    :data="chatData"
  />

  <!-- Con imagen de usuario -->
  <ea-chat
    userIcon="/images/avatar-user.png"
    :data="chatData"
  />
</template>
ts
<script setup lang="ts">
  import { ref } from 'data';
  import { EaChat } from '@zurich-es-npm/ea-front-web-ui';

  const chatData = ref([
    {
      type: 'us',
      text: 'Hola'
    }
  ])
</script>

Botón de cerrar Terminado ​

La propiedad closeButton permite mostrar y ocultar el botón de cerrar, siendo su valor inicial true.

Visibilidad Terminado ​

La propiedad open permite abrir o cerrar el chat, siendo su valor inicial true. Está propiedad junto con el evento @close permiten abrir y cerrar un chat como se mostramos a continuación.
Ver código
vue
<template>
  <ea-chat
    min-button
    :open="open"
    @close="open=false"
  />
  <ea-button @click="open=true">Abrir Chat</ea-button>
</template>
ts
<script setup lang="ts">
  import { EaChat } from '@zurich-es-npm/ea-front-web-ui';

  const open = ref(true);
</script>

Dimensiones Terminado ​

Con las propiedades width y height podemos definir las dimesiones del chat, siendo su valores posibles cualquier unidad css. Por defecto sus valores son de 300px de ancho y 500px de alto.

Importante

Las dimensiones minimas del chat son de 250px de ancho y alto, por lo tanto las propiedas width y height no tendrán efecto por debajo de esos valores.

300x500 (por defecto)
Altura de la mitad del alto de la pantalla
400x500
Se ajusta a su contenedor
Ver código
vue
<template>
  <!-- 300x500 (por defecto) -->
  <ea-chat />
  
  <!-- Altura de la mitad del alto de la pantalla -->
  <ea-chat height="50vh" />
  
  <!-- 400x500 -->
  <ea-chat height="500" width="400px" />
  
  <!-- Se ajusta a su contenedor -->
  <ea-chat height="auto" width="auto" />
</template>
ts
<script setup lang="ts">
  import { EaChat } from '@zurich-es-npm/ea-front-web-ui';
</script>

Añadir respuesta de la inteligencia artificial Terminado ​

Con los métodos addMessage, showLoading y hideLoading, podemos añadir la respuesta de la inteligencia artificial.
Título inicial
Ver código
vue
<template>
  <ea-chat 
    ref="chatRef" 
    @send="handleSend"
  />
</template>
ts
<script setup lang="ts">
  import { EaChat } from '@zurich-es-npm/ea-front-web-ui';
  const chatRef = ref<InstanceType<typeof EaChat>>();

  async function handleSend(message: string) {
    chatRef.value.showLoading();
    await function timeout(ms: number) {
      return new Promise(resolve => setTimeout(resolve, ms));
    }(3000);
    chatRef.value.addMessage('Respuesta automatica de la inteligencia artificial');
    chatRef.value.hideLoading();
  }
</script>

Ubicaciones del chat Terminado ​

La propiedad target permite asignar el chat a un contenedor, página o body. Si no se especifica esta propiedad se ubica en la posición del tag EaChat.
Chat en el cuerpo principal de la aplicación
Ver código
vue
<template>
  <ea-button type="secondary" @click="handleClick">{{ text }}</ea-button>

  <ea-chat 
    type="float" 
    target="body"  
    :open="openChat"
    minButton
  />
</template>
ts
<script setup lang="ts">
  import { ref } from 'vue';
  import { EaChat, EaButton } from '@zurich-es-npm/ea-front-web-ui';

  const openChat = ref(false);
  const text = ref('Mostrar chat flotante en la página');

  function handleClick() {
    openChat.value=!openChat.value;
    text.value = openChat.value ? 'Ocultar chat flotante en la página': 'Mostrar chat flotante en la página';
  }
</script>
Chat en un contenedor
Ver código
vue
<template>
  <div class="container hp-600 t-border-solid-primary-2 " />

    type="float"

    :open="openChat"
    minButton
    bottom="68"
  />
  <ea-button class="m-t-24" type="secondary" @click="handleClick">{{ text }}</ea-button>
</template>
ts
<script setup lang="ts">
  import { ref } from 'vue';
  import { EaChat, EaButton } from '@zurich-es-npm/ea-front-web-ui';

  const openChat = ref(false);
  const text2 = ref('Mostrar chat flotante en el contenedor');

    openChat.value=!openChat.value;

</script>

La propiedad animated permite habilitar y deshabilitar las animaciones del chat, siendo su valores posibles true (por defecto), y false.
Con animación
Sin animación

vue
<template>
