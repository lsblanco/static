  • Se modifica el componente ea-sms para que la OTP funcione correctamente en dispositivos móviles Android e IOS.
  • Se modifica el componente ea-backtop para que las propiedades right, bottom y visibilityHeight, se puedan modificar en dispositivos móviles.
  • Se modifica el componente ea-dialog para incorporar un nuevo tipo fullscreen, ocupando toda la página en dispositivos móviles.

Cambios de diseño: ​

  • Se modifica el diseño de la propiedad disabled en el componente ea-check para que se vea adecuadamente.

Cambios en la documentación: ​

  • Se crea nueva documentación en el componente ea-table que permite visualizar en una columna dos o más líneas con componentes ea-text.
