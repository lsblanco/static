Introducción ​

Los temas permiten cambian el aspecto visual de componentes y patrones que componen el sistema de diseño.
Mediante la personalización de un tema podemos crear un branding de diseño para crear y gestionar la identidad visual de una marca, incluyendo elementos como el logotipo, los colores, la tipografía y otros aspectos visuales que la representan.

Temas disponibles ​

Los temas disponibles en el sistema de diseño son Zurich Light (por defecto), Zurich Dark, auto y custom .
El significado de los temas se detalla a continuación:
  • Zurich Light: El modo claro ambia la paleta de colores para mejorar la visibilidad en condiciones de buena iluminación.
  • Zurich Dark: El modo oscuro cambia la paleta de colores para invertir los colores de los elementos de la interfaz, reduciendo el brillo y evitando que la vista se canse.
  • Auto: Modo claro u oscuro dependiendo del estado o configuración del navegador.
  • Custom: El modo custom permite personalizar un tema creando una marca o branding de diseño en una aplicación. Para más información ver la sección, ¿Cómo crear un tema personalizado?

Instalación ​

Para instalar un tema o marca de diseño, iremos al archivo main.ts o inicio de aplicación, e intalaremos el plugin de temas (themePlugin).
ts
// file main.ts
import { createApp } from 'vue';
import { createI18n } from 'vue-i18n';
import { i18nConfig } from './../index';
import themePlugin from "@zurich-es-npm/ea-front-web-ui"; 


import App from './App.vue';
import './../assets/scss/main.css';

const i18n = createI18n(i18nConfig);
const app = createApp(App);

app.use(i18n);
app.use(themePlugin); 
app.mount('#app');

Cambio de tema ​

Inicialmente el sistema de diseño tiene el tema de color Zurich Light, pero podemos cambiarlo en una aplicación o componente por los temas disponibles.

En el inicio de una aplicación ​

Para cambiar un tema de diseño, instalamos en el inicio de aplicación o main.ts el composable de temas (useTheme).
ts
// file main.ts
import { createApp } from 'vue';
import { createI18n } from 'vue-i18n';
import { i18nConfig } from './../index';
import themePlugin from "@zurich-es-npm/ea-front-web-ui"; 
import { useTheme } from '@zurich-es-npm/ea-front-web-ui/useTheme';


import App from './App.vue';
import './../assets/scss/main.css';

const i18n = createI18n(i18nConfig);
const app = createApp(App);

app.use(i18n);
app.use(themePlugin); 
useTheme().setTheme('dark');
app.mount('#app');

En un componente ​

Con la propiedad theme de cualquier componente podemos elegir uno de los temas disponibles.
vue
<template>
  <!-- Tema light (por defecto) -->
  <ea-alert />
 
  <!-- Tema dark -->
  <ea-alert theme="dark" />
</template>

¿Cómo crear un tema personalizado? ​

Para crear un nuevo tema de diseño personalizado tendremos que realizar los siguientes pasos:

  1. Situarse en al archivo localizado en la ruta src/assests/css/customTheme.css donde incluiremos los cambios del nuevo tema.

  1. Modificar los tokens base y de componentes con el nuevo estilo de marca.
css
[data-theme="custom"] {
  /* Base colors */
  --ea-color-primary: #2167AE;
  ...
  /* Alert component */
  --ea-alert-info-bg-color: var(--ea-color-primary);  
}
  1. Asignar el plugin y el tema en el inicio de la aplicación.
ts
// file main.ts
import { createApp } from 'vue';
import { createI18n } from 'vue-i18n';
import { i18nConfig } from './../index';
import themePlugin from "@zurich-es-npm/ea-front-web-ui"; 
import { useTheme } from '@zurich-es-npm/ea-front-web-ui/useTheme'; 


import App from './App.vue';
import './../assets/scss/main.css';

const i18n = createI18n(i18nConfig);
const app = createApp(App);

app.use(i18n);
app.use(themePlugin); 
useTheme().setTheme('custom'); 
app.mount('#app');

Importante

El sistema de diseño provee de tokens para crear un tema personalizado o branding de marca con facilidad. Disponemos de tokens base para el cambio colores, tipografías, etc, y de Tokens por componentes para el cambio de componentes y patrones.
